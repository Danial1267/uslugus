# WEBPACK

### Для установки пакетов используйте команду
```
npm install
```

## Команды

### Запуск сервера для разработки
```shell
npm run start
```

### Сборка проекта без оптимизации для разработки и тестирования
```shell
npm run build-dev
```

### Сборка проекта с оптимизацией для продакшен версии
```shell
npm run build-prod
```

### Очистка папки dist
```shell
npm run clear
```
